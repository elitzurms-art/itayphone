# ItayPhone — תכנון פרויקט סמארטפון DIY

> מטרה: **סמארטפון מלא** (מסך מגע, אפליקציות, 4G, מצלמה) — אבל עם **מערכת ומאפיינים משלנו**.
> כלומר: בונים טלפון מבוסס לינוקס שבו אנחנו שולטים בכל שכבת התוכנה.

עודכן: 2026-06-29

---

## 1. החלטת ארכיטקטורה

הדרך הריאלית לבנות סמארטפון "אמיתי" עם קוד משלנו, בלי לעצב מעגל מודפס מאפס, היא **מחשב לוח יחיד (SBC) + מודם סלולרי + מסך מגע + סוללה**, והכול רץ לינוקס עם ממשק שאנחנו כותבים.

**פלטפורמה נבחרת: Raspberry Pi 5 (8GB)**
- בחרנו במסלול **לינוקס + Waydroid** (ראה סעיף 4) — שמריץ אנדרואיד בקונטיינר ולכן כבד.
- 8GB RAM + מעבד Pi 5 דרושים כדי שלינוקס + הממשק שלנו + Waydroid ירוצו חלק.
- מחיר: דורש **קירור אקטיבי** (מאוורר רשמי) וצריכת חשמל גבוהה → סוללה גדולה יותר.
- חלופה חסכונית: Pi 4 (4GB) — עובד, אבל Waydroid מגמגם באפליקציות כבדות.

## 3. אתגרים ידועים (חשוב לדעת מראש)

1. **אודיו של שיחות** — החלק הכי קשה. ה-SIM7600 מנתב את אודיו השיחה דרך פיני האודיו שלו (אנלוגי), לא דרך ה-Pi. נחבר רמקול+מיק לפיני/שקע האודיו של ה-HAT.
2. **צריכת חשמל וחום** — Pi 4 חם; צריך קירור וניהול סוללה טוב. זמן סוללה צפוי: שעות בודדות בשימוש פעיל.
3. **מסך GPIO vs HAT** — לכן בחרנו מסך DSI (ולא DPI/Hyperpixel) כדי לא להתנגש עם ה-4G HAT.
4. **גודל ומשקל** — זה יהיה "סמארטפון עבה". פרוטוטיפ ראשון יהיה על שולחן, לא בכיס.

## 4. ערימת התוכנה (Software Stack) — "הפיצ'רים משלנו"

- **OS בסיס:** Raspberry Pi OS (Lite או Desktop) / או Wayland מינימלי.
- **שכבת תקשורת:** דמון Python שמדבר עם ה-SIM7600 ב-AT commands (שיחות, SMS, רשת).
- **ממשק (UI/Launcher):** אפליקציית מסך-מלא שאנחנו כותבים. מועמדים:
  - **Python + Kivy** — הכי מהיר לפיתוח מסך-מגע, מתאים לפיצ'רים מותאמים.
  - חלופה: PyQt / QML, או web-app מקומי ב-Chromium kiosk.
- **אפליקציות הליבה שלנו:** חייגן, SMS, אנשי קשר, מצלמה, מסך בית (launcher) — כולן הקוד שלנו.
- **Waydroid (נבחר):** אנדרואיד בקונטיינר על גבי הלינוקס שלנו → להריץ אפליקציות אנדרואיד (וואטסאפ וכו') לצד הממשק שלנו.
  - דורש OS 64-bit עם תמיכת kernel ב-binder (Raspberry Pi OS 64-bit / Ubuntu).
  - שיחות ו-SMS אמיתיים נשארים דרך הקוד שלנו + SIM7600 (לא דרך אנדרואיד).
- **שלד הקוד:** ריפו Python מודולרי — מודול לכל תת-מערכת.
