# ItayPhone על אנדרואיד 📱

אריזה של האפליקציה ל‑APK שאפשר להתקין על טלפון אנדרואיד.

## מה עובד באמת על אנדרואיד

| פיצ'ר | התנהגות על אנדרואיד |
|-------|----------------------|
| הפעלת אפליקציות (WhatsApp / Telegram / Chrome) | **פותח את האפליקציה האמיתית** המותקנת בטלפון (ואם לא מותקנת — את חנות Play) |
| חיוג | פותח את מסך החיוג של אנדרואיד עם המספר |
| SMS | פותח את אפליקציית ההודעות עם הנמען |
| פנס | **הפנס האמיתי** של המצלמה |
| מצלמה | מצלם דרך מצלמת המערכת |
| Wi‑Fi / בלוטות' | פותח את פאנל ההגדרות של אנדרואיד (אנדרואיד חוסם סריקה/כיבוי שקטים לאפליקציות צד‑שלישי) |
| הממשק כולו, גלריה, מרכז בקרה, מחוות | עובד מקומית |

> השיחות/SMS עוברות דרך אפליקציות המערכת של אנדרואיד — אין רדיו SIM7600 בטלפון רגיל.

## איך בונים APK (בענן, בלי Linux מקומי)

הפרויקט כולל workflow של **GitHub Actions** (`.github/workflows/build-apk.yml`):

1. דוחפים את הפרויקט ל‑GitHub (פרטי).
2. ה‑Action בונה אוטומטית `itayphone-0.1-debug.apk` (~20–30 דק').
3. מורידים את ה‑APK מ‑**Actions → הריצה האחרונה → Artifacts**.
4. מעבירים לטלפון, פותחים, מאשרים "התקנה ממקור לא ידוע", מתקינים.

## בנייה מקומית (אם יש Linux / WSL)

```bash
pip install buildozer
buildozer android debug      # הפלט: bin/itayphone-0.1-debug.apk
```

## הרצה במחשב (תצוגה מקדימה, מצב mock)

```bash
python main.py
```
